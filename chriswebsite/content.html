Content management systems

System Produced/Sold by Technology Base Capabilities Limitations Example Website
phpWebLog Jason Hines MySQL Story moderation, threaded comments, templating/themes, polls, multi-language translations, RDF implication Meant for web blogs, last stable release was in 2013. https://sourceforge.net/p/phpweblog/
Fedora Commons DuraSpace MySQL, Oracle, PostgreSQL, Mulgara (MPTSTore RDF Semantic Triplestore) Open Source, highend security Not meant for those technically inclined with Linux. https://duraspace.org/fedora/
Wiki.js Nicolas Giard and community members Javascript, MongoDB Markdown editor,Git-backed storage, Assests Managment and Built-in search engine. Must have the lastest version of Node.js and MongoDB in order to run a Wiki.js server https://wiki.js.org/
Joomla! Open Source Matters, Inc. and the community PHP, MySQL, MSSQL, SQLite, Cross-platform Widly used, supports up to 72 languages Harder to use than most other websites https://www.joomla.org/
WoodPress WordPress Foundation PHP, MySQL, MariaDB Very user friendly, large community, supports plugins and themes Plugins and themes can be vulnerable to attacks if not patched correctly https://wordpress.org/
XOOPS The XOOPS Project PHP, MySQL Free, open source and user friendly Not as popular as other management systems https://xoops.org/
