Bachelor of Science in Information and Technology

BSIT (6334) Advising guide


Information:

This advising guild is used to help students, who are in the BSIT program. Everything is color coded and labled to make class selection easy and simple.

Key:

Classes are highlighted in three different colors.

Yellow means you have not taken the class yet and is ready to be taken.

Red means the class needs a pre-requisite for it to be taken. Some require two classes to fill in this requirement. However, if the requirment only allows for one class type to be taken, then it will aoutmatically be disabled in red.

Green and in bold means the class is currently being taken or as been taken.

Course color coding:

These classes are seperated into boxes, with each box having a background color.

Gray is prevoiusly taken classes.

Blue is math classes.

Light Green is Linux classes.

Light Brown is programming classes.

Light Red is electronics classes.

Light Orange is general IT classes.

Light Purple is the cybersecurity class.
